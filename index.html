Data acquisition


The data acquisition should accurately and at right moment measure all the measurands. Then the measurement data has to be communicated to a central node where the processing is done. This is a separate high-level function. This communication protocol should be redundant and has to be selfaware of any errors made in transmission. Besides that the acquisition should be low power and protected against the environment.

Data Processing


The data from the sensors should be combined with data from outside sources and processed into useful information for the farmer. The system should also be able to detect and correct for any failures in the system.

Communicaiton data


The system needs to communicate different data sets and predictions to the farmer in different ways. This top-level function deviates in sub-sets of what data needs to be communicated to the farmer and in what way to make it an easy to understand system.
